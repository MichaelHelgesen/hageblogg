────────────────────────────────────────────────────────────────────────────────
POST (Blogginnlegg)
────────────────────────────────────────────────────────────────────────────────
Fil: sanity/schemaTypes/post.ts
URL: /blogg/{slug}

FELTER:
  title*            string        Tittel
  slug*             slug          URL-vennlig identifikator
  mainImage         image         Hovedbilde
    ├─ alt*         string        Alt-tekst
    ├─ caption      string        Bildetittel
    └─ description  text          Bildetekst/Beskrivelse
  gallery           image[]       Bildegalleri (samme underfelter som mainImage)
  categories*       reference[]   Kategorier (min. 1 påkrevd) → category
  plantTypes        reference[]   Plantetyper → plantType
  plants            reference[]   Planter/sorter → plant
  listText          text          Kort tekst for lister
  content           blockContent  Hovedinnhold (rik tekst)
  relatedPosts      reference[]   Relaterte innlegg → post
  relatedGuides     reference[]   Relaterte guider → guide
  relatedPlants     reference[]   Relaterte planter → plant

PLANTEHIERARKI
================================================================================

  plantCategory (Plantekategori)
       │
       └── plantType (Plantetype)
              │
              └── plant (Plante/Sort)

Eksempel:
  Grønnsaker (kategori)
       └── Tomat (type)
              ├── San Marzano (sort)
              └── Cherry (sort)

BLOCKCONTENT (Rik tekst-innhold)
================================================================================
Fil: sanity/schemaTypes/blockContent.ts

Brukes i: post.content, page.content, guide.steps[].content

STANDARD BLOKKER:
  block             Vanlig tekst med formatering
  image             Bilder med alt-tekst og caption

CUSTOM BLOKKER:
  ┌─────────────────────────────────────────────────────────────────────────────
  │ tipBlock (Tips-blokk)
  │ Fremhevet tekstboks for tips og advarsler
  ├─────────────────────────────────────────────────────────────────────────────
  │   variant*      string        "plant" | "tool" | "remember" | "warning"
  │   title         string        Tittel
  │   tip           block[]       Innhold (rik tekst)
  └─────────────────────────────────────────────────────────────────────────────

  ┌─────────────────────────────────────────────────────────────────────────────
  │ relatedGuide (Relatert guide)
  │ Referanse til en eksisterende guide
  ├─────────────────────────────────────────────────────────────────────────────
  │   guide         reference     Referanse → guide
  │   showDescription boolean     Vis beskrivelse
  │   displayMode   string        "short" | "full" | "collapsed"
  └─────────────────────────────────────────────────────────────────────────────

  ┌─────────────────────────────────────────────────────────────────────────────
  │ relatedPost (Relatert innlegg)
  │ Referanse til et blogginnlegg
  ├─────────────────────────────────────────────────────────────────────────────
  │   post*         reference     Referanse → post
  └─────────────────────────────────────────────────────────────────────────────

  ┌─────────────────────────────────────────────────────────────────────────────
  │ stepGuide (Intern steg-for-steg guide)
  │ Guide som kun lever i blogginnlegget
  ├─────────────────────────────────────────────────────────────────────────────
  │   title         string        Tittel på guiden
  │   steps         step[]        Steg
  │     ├─ title    string        Stegtittel
  │     ├─ content  block[]       Innhold (tekst + bilder)
  │     └─ image    image         Hovedbilde for steget
  └─────────────────────────────────────────────────────────────────────────────

MARKS (Inline-formatering):
  strong            Fet tekst
  em                Kursiv tekst
  link              Ekstern lenke (href)
  internalLink      Intern lenke → post, plant, category, guide
